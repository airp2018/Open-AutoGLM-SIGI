## 01. 这是什么？(What is SIGI?)

> **“在末日之前，再踩一脚油门。”**

这不仅仅是一个 App，这是一个 **思想装置艺术**，也是一台能帮你操控现实的 **“赛博机器”**。

它的灵感来源于 **《三体》 (The Three-Body Problem)** 与 **古典赛博朋克**。
在这个 App 里，我们将世界简化为两个按钮，就像开车一样：

*   🔴 **EXECUTE (油门)**：代表 **加速主义 (e/acc)**。按下它，AI 将接管你的手机，全速执行任务，冲向未知的技术奇点。这就像《黑客帝国》里的红色药丸，或是密室逃脱的终极开关。
*   🟢 **ABORT (刹车)**：代表 **拯救派 (Salvation)**。按下它，切断连接，你便拯救了世界（暂时）。
*   ⌨️ **INPUT (方向盘)**：你输入的提示词，决定了我们要驶向何方。

这台 “**豆包手机**” 不需要你花一分钱购买新硬件。
它就在你的旧 Android 手机里，只要你植入 SIGI，它就变成了 **The Last Cyberphone**。

3.  **末日清单 (Doomsday List)**：把你最想做的事存进去——比如“给喜欢的人发一句晚安”、“抢一张回家的车票”、“点一份最后的肯德基”。

无需 Root，无需复杂的电脑编程。

## 03. 极简安装指南 (小白必看)

不要被“代码”吓跑。我们为你准备了 **[一键安装包]**。

### ✅ 零基础安装法 (推荐)
**前期准备**：一台电脑 (Windows)，一根数据线，你的安卓手机。

1.  **下载**：获取 `AutoGLM-SIGI-一键安装包.zip` 并解压。
2.  **连线**：
    *   手机开启 **[USB调试]** (通常在 设置 -> 关于手机 -> 连续点击版本号7次 -> 开发者选项 里)。
    *   插上数据线连接电脑。
3.  **双击**：
    *   打开文件夹，双击运行 **`一键安装(双击我).bat`**。
    *   看着屏幕出现的绿色文字，等待提示“安装成功”。
4.  **开始**：拔掉线，打开手机上的 **SIGI** 图标。

### 💻 极客安装法 (GitHub)
如果你是开发者，欢迎访问 GitHub 源码：
`git clone https://github.com/ToTheMoon/Open-AutoGLM-SIGI.git`
我们保留了最纯粹的 ADB 权限接口，供你探索 Android 的极限。

## 04. 架构原理 (为什么它这么强？)

市面上的“AI 手机”动辄几千块，其实核心就是 **LAM (大动作模型)** 技术。
而 SIGI 为你接入了开源界最强的 **AutoGLM** 引擎。
SIGI 采用了 **Hybrid 混合架构**：

`[你的指令] -> [SIGI 赛博终端] -> [AutoGLM 大脑] -> [ADB 底层权限] -> [操控手机]`

*   **无需一直插电脑**：只有安装时需要电脑授权一次，之后它就是一台独立的 AI 手机。
*   **视觉识别**：它不是在瞎点，它是真的“看见”了屏幕上的字。
