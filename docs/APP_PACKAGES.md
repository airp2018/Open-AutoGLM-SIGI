## 🔍 包名查询命令

```bash
# 查看所有已安装的包
adb shell pm list packages

# 搜索特定 APP
adb shell pm list packages | findstr -i "关键词"

# 查看 APP 详细信息
adb shell dumpsys package 包名 | findstr -i "version"

# 启动 APP
adb shell am start -n 包名/.MainActivity
```
